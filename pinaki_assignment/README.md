# AI Document Generator

A full-stack, AI-powered web application for generating, refining, and exporting structured business documents.

## Features

- User Authentication with JWT
- AI Content Generation using Google Gemini
- Word (.docx) and PowerPoint (.pptx) export
- Real-time editor with auto-save
- AI-suggested outlines
- Professional document formatting

## Tech Stack

- Flask, SQLAlchemy, MySQL
- HTML, CSS, JavaScript, Bootstrap 5
- Google Gemini AI API
- python-docx, python-pptx

## Quick Start

1. Clone and install dependencies
2. Set up environment variables
3. Run `python app.py` in app/ directory
4. Access at http://localhost:5000
